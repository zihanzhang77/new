# AI图片生成器

一个简单的AI图片生成网站，使用Coze API生成图片。

## 功能特点

- 简洁的用户界面
- 实时图片生成
- 支持在新标签页中查看生成的图片

## 使用方法

1. 在输入框中输入提示词
2. 点击"立刻生成"按钮
3. 等待图片生成
4. 点击链接在新标签页中查看生成的图片

## 技术栈

- HTML5
- CSS3
- JavaScript
- Coze API
